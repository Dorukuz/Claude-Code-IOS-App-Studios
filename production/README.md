# Production tracking

- **`sprints/`** — sprint plans (`sprint-01.md`, …)
- **`milestones/`** — milestone specs
- **`session-state/`** — optional `active.md` for in-progress context (see hooks)
- **`session-logs/`** — generated logs (gitignored optionally per team policy)

Do not commit secrets, API keys, or provisioning profiles here.
